## 🚀 Overview

**Neurolight Galactic** is an AI Content Detector with a whimsical, comic-inspired frontend and a powerful Hugging Face-backed FastAPI backend. It allows users to scan **Text, Images, and Videos** to determine whether the content was generated by AI or created by an authentic human.

### ✨ Features
- **Text Assay**: Detects AI-generated text (ChatGPT, etc.) using the robust `Hello-SimpleAI/chatgpt-detector-roberta` model.
- **Image Assay**: Identifies AI-generated deepfakes and images utilizing `dima806/deepfake_vs_real_image_detection`.
- **Video Assay**: Efficiently analyzes video payloads by extracting and scanning strategic frames (at 33% and 66% marks) via OpenCV, bypassing heavy video processing overhead.
- **Cosmic UI**: A vibrant, retro-futuristic interface designed for an engaging user experience.

---

## 🏗️ Architecture

The project is split into two distinct parts:

1. **Backend (`/backend`)**: A robust FastAPI application that acts as a proxy to Hugging Face's Inference APIs. It parses multipart form data, chunks video frames using OpenCV, and safely processes Hugging Face responses.
2. **Frontend (`/frontend`)**: A pure HTML/CSS/JS client that interacts with the backend. It features dynamic tab switching, file uploads, and animated result displays.

## 🛠️ Local Setup

Follow these instructions to spin up the *Galactic Core* on your local machine.

### Prerequisites
- Python 3.8+
- A Hugging Face account and Access Token

### 1. Clone the repository
```bash
git clone https://github.com/ParvathiDev23/ai-content-detector.git
cd ai-content-detector
```

### 2. Backend Setup
```bash
# Navigate to the backend
cd backend

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate

# Install dependencies
pip install -r ../requirements.txt

# Set your Hugging Face Token as an environment variable
export HF_TOKEN="your_hugging_face_token_here"

# Run the FastAPI server
uvicorn main:app --reload --port 8000
```
*The backend will now be running on `http://localhost:8000`.*

### 3. Frontend Setup
1. Open `frontend/index.html` in your favorite text editor.
2. Ensure the `API_BASE_URL` on line 178 points to your local backend (e.g., `http://localhost:8000`) for local testing.
3. Simply open `index.html` in any web browser!

## 📡 API Endpoints

- `GET /` - Health check. Returns `{"status": "Strict Binary API is running!"}`
- `POST /analyze-text` - Accepts JSON `{"text": "your text here"}`.
- `POST /analyze-image` - Accepts a multipart form data file upload (`image/*`).
- `POST /analyze-video` - Accepts a multipart form data file upload (`video/*`).
